# Rekomendacijos užduočių kodo išsisaugojimui

Rekomenduojama kiekvieną pavyzdį (projektą) saugoti skirtinguose aplankuose, kadangi projekte gali būti keli failai. Pavyzdžiui:

```
htdocs/
    pavyzdys1/
        index.php
    pavyzdys2/
        index.php
    pavyzdys3/
        klase.php
        index.php
    pavyzdys4/
        index.php
        automobilis.php
    pavyzdys5/
        index.php
        studentas.php
        grupe.php
        universitetas.php
    projektas1/
        index.php
        automobilis.php
        servisas.php
        klientas.php
    parduotuve/
        index.php
        preke.php
        parduotuve.php
        pirkejas.php
        krepselis.php
        ...
    ...
```
